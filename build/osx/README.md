# Alpaca

![Latest Tag][2] ![GitHub Workflow Status][3] ![GitHub Releases][4]

Alpaca is a local HTTP proxy for command-line tools. It supports proxy
auto-configuration (PAC) files and NTLM authentication.

## Build macOS paclage from source

The commands below will download latest alpaca and create a macOS package from it.

```sh
$ sh build-pkg-from-src-mac
```

Once built you can open the package in finder to install it.
